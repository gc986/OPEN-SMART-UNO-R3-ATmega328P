<h3>Распиновка</h3>
<b>Кнопки:</b>
<ul>
  <li>Key1 - D4</li>
  <li>Key2 - D3</li>
  <li>Key3 - D2</li>
</ul>

<b>Светодиоды:</b>
<ul>
  <li>LED1 - D13</li>
  <li>LED2 - D8</li>
  <li>LED3 - D7</li>
</ul>

<b>Зумер - D6</b>

Всё остальное как на обычной "Arduino Uno"
